# Rastreio de Telefone

Aplicação em Python (Tkinter) para consulta de informações públicas de números de telefone.

## O que o app faz
- Mostra o país do número
- Mosttra a região aproximada do prefixo
- Mostra a operadora (quando disponível)
- Mostra o horário local aproximado

> Este projeto não rastreia localização em tempo real.  
> As informações são baseadas em metadados públicos do número.
